\doxysubsection{Detailed Description}
A pipeline element to derive real-\/time noise removal filters from. Inherits \mbox{\hyperlink{classSampleLink}{Sample\+Link}}. Derived Classes must implement the pure abstract \mbox{\hyperlink{classNoiseFilter_af7eb0fd8ea4b9656379cdffa8d170ad7}{Filter()}} function. Noise and dirty samples are synchronised by making use of two instances of \mbox{\hyperlink{classBlockingQueue}{Blocking\+Queue}}. A worker thread loads each samples pair and performs the filtering per-\/sample.
